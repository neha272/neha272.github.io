# Neha Sutariya - Portfolio

Professional portfolio website built with Bootstrap 4, showcasing skills, experience, education, and projects.

## Features

- Responsive design
- Modern UI/UX
- Smooth scrolling navigation
- Skills showcase with icons
- Project portfolio
- Education and experience timeline

## Setup

### Prerequisites

- Node.js and npm (optional - vendor files are already included)

### Installation

If you want to rebuild assets:

```bash
npm install
gulp
```

### Development

To run with live reload (if Gulp is set up):

```bash
gulp dev
```

Then open `index.html` in your browser or use a local server.

## Deployment

This portfolio is designed to be deployed on GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in repository settings.

## Customization

Update the following files to customize:
- `index.html` - Main content and structure
- `scss/` - Styles and theming
- `img/profile.jpg` - Profile picture
